Ce chat se lance en étant placé dans le répertoir du projet "Proj_SE-master". 
Une fois placé, compiler les programmes en tapant la commande "make all".

Le programme est maintenant compilé, et les deux exécutables "client" et "serveur" ont été créés.

Lancer le contrôleur en tapant la commande "./serveur", puis lancer un ou plusieurs client(s) dans une nouvelle (des nouvelles) console(s) en tapant "./client".
Il est possible de lancer directement le client avec "./client", ce qui lancera automatiquement un contrôleur.

Toutes les commmandes accessibles sont affichés sur les consoles. Vous pouvez maintenant utiliser le chat en tapant vos messages dans les consoles des clients !
